### CSS 改变input placeholder 颜色
```
input:-ms-input-placeholder {
    color: #a9a9a9;
}/* Internet Explorer 10+ */

input::-webkit-input-placeholder {
    color: #a9a9a9;
}/* WebKit browsers */

input::-moz-placeholder {
    color: #a9a9a9;
}/* Mozilla Firefox 4 to 18 */

input:-moz-placeholder {
    color: #a9a9a9;
}/* Mozilla Firefox 19+ */
```
